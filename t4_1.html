1.4 Hypoth√®ses g√©n√©rales d‚Äôint√©r√™t

  1. Existe-t-il une interaction entre les facteurs A et B ?

\[ H_{0AB} : \mu_{ij} - \mu_{im} - \mu_{lj} + \mu_{lm} = 0 \quad \text{pour tout } (i,j,l,m) \quad \text{(pas d'interaction)} \]
\[ H_{1AB} : \mu_{ij} - \mu_{im} - \mu_{lj} + \mu_{lm} \neq 0 \quad \text{pour au moins un } (i,j,l,m) \quad \text{(interaction)} \]

  1. Test sur l‚Äôeffet global du facteur A

\[ H_{0A} : \mu_{1\bullet} = \mu_{2\bullet} = \cdots = \mu_{a\bullet} \]
\[ H_{1A} : \text{les } \mu_{i\bullet} \text{ ne sont pas tous √©gaux.} \]
(l‚Äôeffet global du facteur A est significatif)

  1. Test sur l‚Äôeffet global du facteur B

\[ H_{0B} : \mu_{\bullet 1} = \mu_{\bullet 2} = \cdots = \mu_{\bullet b} \]
\[ H_{1B} : \text{les } \mu_{\bullet j} \text{ ne sont pas tous √©gaux.} \]
(l‚Äôeffet global du facteur B est significatif)

1.5 M√©thodologie

On commence par tester s‚Äôil y a interaction (H0AB)
